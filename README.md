# 🎮 Arceus Clicker Django Edition

> **Projeto de estudo desenvolvido para explorar o ecossistema Full-Stack com Python e Django.**

O **Arceus Clicker** é um jogo experimental que une a nostalgia da franquia Pokémon com a mecânica viciante dos jogos *idle/clicker*. O projeto foi concebido como um estudo de caso para praticar o desenvolvimento web, gerenciamento de estados no servidor e integração de bancos de dados.

---

## 📋 Sobre o Projeto

O objetivo principal é permitir que o usuário avance em sua jornada Pokémon através de cliques interativos. A cada interação, o jogador acumula recursos, sobe de nível e desbloqueia novos elementos, tudo processado de forma robusta pelo backend.

### 🛠️ Tecnologias Utilizadas

* **Backend:** [Python](https://www.python.org/) + [Django Framework](https://www.djangoproject.com/)
* **Frontend:** HTML5, CSS3 e JavaScript (para atualizações dinâmicas).
* **Banco de Dados:** SQLite (Desenvolvimento).
* **Arquitetura:** MVT (Model-View-Template).
